## Gotchas
- Two Introspect processes: installed /Applications (old) vs .build (new). Must install to see changes.
  - Install = `ditto .build/Introspect.app /Applications/Introspect.app` after build-introspect-app.sh. `open -a Introspect` resolves to /Applications, so a stale copy there shows old UI even when .build is newer.
  - Verify visually: AX window bounds are blocked, so use `screencapture -x -o` then `magick … -crop 2480x1640+488+198` (window opens 1240×820 after `defaults delete ai.companion.introspect "NSWindow Frame IntrospectMainWindow"`). Arc/other windows overlap + steal frontmost; activate Introspect right before each capture.
- `version` field on events = app-repo git short hash (join to commit subject works).
- Triggering happens in hooks/codex-transcript-scan.py (regex word match); NL sentiment is display-only.
